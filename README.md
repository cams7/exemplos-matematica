Qual a finalidade desses exemplos?
-------------------
O objetivo é aprender um pouco mais sobre cálculos de limites, derivadas e integrais.

Sistemas requeridos
-------------------
* [Matlab R2014A] (http://www.mathworks.com/products/matlab/)
